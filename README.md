# ClassHub - 학급 종합 관리 시스템 🎓

> 대한민국 학교 현장을 위한 올인원 학급 관리 솔루션

[![GitHub Pages](https://img.shields.io/badge/demo-live-success)](https://your-username.github.io/school_revolution_prototype)
[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](LICENSE)

## 📌 소개

ClassHub는 학생과 교사가 학교 생활에 필요한 모든 기능을 하나의 플랫폼에서 이용할 수 있도록 설계된 정적 웹 애플리케이션입니다. GitHub Pages를 통해 무료로 호스팅 가능하며, 별도의 서버나 데이터베이스 없이 브라우저의 LocalStorage를 활용하여 모든 데이터를 안전하게 저장합니다.

## ✨ 주요 기능

### 📅 스마트 시간표
- 요일별 시간표 관리 (월~금)
- 현재 진행 중인 수업 자동 하이라이트
- 과목, 교사, 시간 정보 관리
- 직관적인 편집 인터페이스

### ⏰ D-Day 카운터
- 중요한 학사 일정 카운트다운 (수능, 시험, 방학 등)
- 다양한 색상 테마로 이벤트 구분
- 자동 날짜 계산 (D-Day, D+Day)
- 무제한 이벤트 추가 가능

### 🍽️ 급식 메뉴
- 주간 급식 메뉴 한눈에 보기
- 오늘 급식 자동 하이라이트
- 요일별 메뉴 관리

### ⏱️ 학습 타이머
- 뽀모도로 기법 지원 (25분 집중 + 5분 휴식)
- 커스터마이징 가능한 시간 설정
- 일일/주간 학습 통계
- 알림 기능

### 📊 성적 계산기
- **내신 계산**: 총 인원과 등수로 내신 등급 자동 계산
- **수능 계산**: 원점수, 평균, 표준편차로 표준점수 및 등급 예측
- 백분위 자동 계산
- 직관적인 결과 표시

### 📢 공지사항
- 중요 공지 강조 표시
- 날짜별 정렬
- 빠른 추가/삭제

### 🎯 할일 관리
- 과제 및 준비물 체크리스트
- 완료율 통계
- 드래그 없이 간편한 체크

### 🌓 다크모드
- 눈의 피로를 줄이는 다크 테마
- 원클릭 테마 전환
- 설정 자동 저장

### 💾 데이터 백업
- 모든 데이터 JSON 파일로 내보내기
- 다른 기기에서 데이터 가져오기
- 데이터 손실 걱정 없음

## 🚀 시작하기

### 온라인에서 바로 사용

1. **GitHub Pages로 배포하기**
   ```bash
   # 1. 이 저장소를 Fork 하세요
   # 2. 저장소 Settings > Pages로 이동
   # 3. Source를 "main" 브랜치로 설정
   # 4. https://your-username.github.io/school_revolution_prototype 에서 접속
   ```

### 로컬에서 실행

1. **저장소 클론**
   ```bash
   git clone https://github.com/your-username/school_revolution_prototype.git
   cd school_revolution_prototype
   ```

2. **로컬 서버 실행**
   ```bash
   # Python 3가 설치되어 있는 경우
   python -m http.server 8000

   # Node.js가 설치되어 있는 경우
   npx serve

   # VS Code Live Server 확장 사용
   # index.html 우클릭 > Open with Live Server
   ```

3. **브라우저에서 열기**
   ```
   http://localhost:8000
   ```

## 📖 사용 방법

### 시간표 등록하기
1. "시간표" 섹션의 **편집** 버튼 클릭
2. 요일별로 과목명과 교사명 입력
3. **저장** 버튼으로 완료

### D-Day 추가하기
1. "D-Day" 섹션의 **+ 추가** 버튼 클릭
2. 이벤트 이름과 날짜 입력
3. 원하는 색상 선택
4. **저장** 버튼으로 완료

### 학습 타이머 사용하기
1. 필요시 집중/휴식 시간 조정
2. **시작** 버튼으로 타이머 시작
3. 타이머 완료 시 자동으로 모드 전환
4. 일일/주간 통계로 학습량 확인

### 성적 계산하기
1. **내신** 또는 **수능** 탭 선택
2. 필요한 정보 입력 (인원, 등수 또는 점수)
3. **계산하기** 버튼 클릭
4. 결과 확인

### 데이터 백업하기
1. 하단의 **데이터 내보내기** 클릭
2. JSON 파일 저장
3. 다른 기기에서 **데이터 가져오기**로 복원

## 🔒 개인정보 보호

ClassHub는 **100% 클라이언트 사이드 애플리케이션**입니다:
- 모든 데이터는 사용자의 브라우저에만 저장됩니다
- 서버로 전송되는 데이터가 없습니다
- 별도의 회원가입이나 로그인이 필요 없습니다
- 데이터 수집이나 추적을 하지 않습니다

⚠️ **주의**: 브라우저 데이터를 삭제하면 모든 정보가 사라지므로, 주기적으로 데이터를 내보내기하는 것을 권장합니다.
